JSP语法-JSP中的指令标识

发表于 2018-07-20 | 更新于 2018-07-25 | 分类于 JSP学习 | 阅读次数：

指令标识概述

  • 指令标识是被服务器解释并且执行的。通过指令标识可以使服务器按照指令的设置来执行动作和设置在整个JSP页面范围内有效的属性。
    指令通常以<%@开始，以%>结束
  • 在JSP中主要包含3种指令，分别是：
    1. page
    2. include
    3. taglib

      这三种指令的通常格式为：

      <%@ 指令名称 属性1="属性值" 属性2="属性值" ...%>

      page指令

  • page指令就是页面指令，可以定义在整个JSP页面范围内有效的属性，其格式如下：

    <%@ page attribute1="value1" attribute2="value2" ...%>

include指令

  • 用于在当前JSP页面中，在当前使用该指令的位置嵌入其他文件，指令格式如下：

    <%@ include file="文件的绝对路径或相对路径" %>

taglib指令

  • 在JSP页面中，可以直接使用JSP提供的一些动作元素标识来完成特定的功能，如使用jsp:include包含一个文件。通过使用taglib指令，就可以在页面中使用这些基本标识或自定义的标识来完成特殊的功能，其指令格式如下：

    <%@ taglib uri="tagURI" prefix="tagPrefix" %>
